## Running the Competition

1.  Setup a python virual environment.

        python -m venv venv

2.  Activate the virtual environment.

3.  Install the required libraries.

        pip install -r requirements.txt

4.  Run the competition.

        python __main__.py

    or

        cd ..

        python resistance

5.  Enjoy.
